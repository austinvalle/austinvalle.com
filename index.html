I design and build software.

I'm Austin Valle, full stack developer currently working at Progressive Insurance. In my current role I lead a small team of developers, building solutions for the Claims platform. These solutions range from web applications to support internal loss reporting, to exposing APIs to partners such as Uber and Safelite. These applications are primarily built with Angular, .NET Web API, and SQL Server/DB2.

In my free time, I enjoy learning new programming languages, frameworks, and emerging technologies. My attention is currently set on Go, Serverless architectures, and React.
